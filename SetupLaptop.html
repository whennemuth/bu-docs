    3. If you find that git bash is behaving slowly for each command on your windows system, this is probably due to UAC (User Account Control).

      1. To fix this issue, you must run git and gitbash as an administrator.

      2. Configure the following file to always run under the administrator account:

      3. c:\[path to git install]\bin\sh.exe

  1. Install Mysql Database (manual install):

    1. Got to http://dev.mysql.com/downloads/mysql/

    • Download Windows (x86, 64-bit), ZIP Archive 5.7.11 336.7M

    1. Once downloaded and unzipped, open a command prompt and cd into the root directory of mysql.

    2. Run the following: bin/mysqld –initialize-insecure
      This command will create the data directory and create a root user WITHOUT a password (use --initialize for password requirement).
      (See mysql system variables at: https://dev.mysql.com/doc/refman/5.6/en/server-system-variables.html)

    3. Create a bat file to start mysql. Contents as follows:

C:\whennemuth\downloads\mysql-5.7.11-winx64\bin\mysqld -u root ^

-console ^

--join_buffer_size=524288 ^

--key_buffer_size=16777216 ^

--read_buffer_size=262144 ^

--read_rnd_buffer_size=524288 ^

--sort_buffer_size=524288 ^

--table_definition_cache=2800 ^

--table_open_cache=4000 ^

--innodb_buffer_pool_size=268435456 ^

--query_cache_size=2097152


    1. Create a bat file to stop mysql. Contents as follows:

C:\whennemuth\downloads\mysql-5.7.11-winx64\bin\mysqladmin -u root shutdown

    1. Add the mysql bin directory to the system path environment variable.
